# Seed Exercise Library

The starter exercise library ships in:

- `Setra/PreviewSupport/SeedData.swift`

It includes coverage for:

- chest
- back
- shoulders
- biceps
- triceps
- forearms
- legs
- calves
- abs / core

Each seeded exercise includes:

- canonical name
- aliases
- primary muscle
- secondary muscles
- equipment
- notes
- rep suggestion
- default set count
- rest time
- progression rule

Included examples:

- Barbell Back Squat
- Romanian Deadlift
- Leg Press
- Lying Leg Curl
- Standing Calf Raise
- Barbell Bench Press
- Incline Dumbbell Press
- Machine Chest Press
- Cable Fly
- Lat Pulldown
- Chest-Supported Row
- One-Arm Dumbbell Row
- Seated Cable Row
- Standing Overhead Press
- Dumbbell Lateral Raise
- Rear Delt Fly
- Barbell Curl
- Incline Dumbbell Curl
- Hammer Curl
- Cable Pushdown
- Overhead Cable Triceps Extension
- Close-Grip Bench Press
- Wrist Curl
- Hanging Leg Raise
- Ab Wheel Rollout
- Weighted Plank

Search behavior is driven by:

- exact name match
- aliases
- fuzzy typo-tolerant ranking
- favorites boost
- recent exercise boost
