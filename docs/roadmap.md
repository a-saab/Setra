# Setra Roadmap

## Phase 1.1

- Add Firebase packages directly to the Xcode project
- Finish Google Sign In URL-type setup
- Expand custom exercise image syncing with Firebase Storage
- Add delete-template and apply-template flows
- Improve set-row ergonomics with swipe actions and quick presets

## Phase 1.2

- Real in-progress workout draft autosave
- Conflict-aware Firestore merge strategy
- More granular workout analytics
- Exercise notes at set level in the live logger
- CSV export model and export UI

## Phase 1.3

- Supersets and drop sets
- Warm-up calculator presets
- Exercise swap during an active session
- Local notifications for rest timer and workout reminders
- Better PR badges and trend callouts

## Phase 2

- Real-time sync listeners across devices
- Apple Sign In
- Watch companion ideas
- Smarter progression modes for bodyweight and assisted movements
- Richer chart customization

## Explicit non-goals preserved

- no social feed
- no subscriptions in v1
- no AI coaching chat
- no meal tracking
- no Cloudflare dependency for v1
