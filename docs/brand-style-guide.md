# Setra Brand Style Guide

## Naming

Initial naming directions considered:

- Setra
- Vanta Lift
- Slate
- Coreform
- Stride Bar

Final selected name:

- `Setra`

Why it works:

- short and memorable
- clean and modern
- feels technical without sounding cold
- strong enough for a premium lifting product

## Brand idea

Setra should feel like:

- precision
- structure
- momentum
- calm confidence

It should not feel:

- cartoonish
- aggressive-bodybuilding loud
- playful-gamified
- neon-supplement brand

## Logo concept

Source of truth:

- `Setra/DesignSystem/SetraTheme.swift`
- `BrandMark`

The in-app mark uses:

- a rounded square glass container
- a diagonal barbell-inspired internal glyph
- one strong icy blue bar plus one white plate-like counterweight

Conceptually it reads as:

- a stylized `S`
- a barbell in motion
- a clean technical mark suitable for an app icon

## App icon concept

Use the `BrandMark` composition on a deep graphite background.

Recommended icon production spec:

- background: radial or linear graphite gradient
- foreground mark: blue-to-steel gradient
- high-contrast white counter-shape
- subtle inner glow, no heavy borders
- rounded square, not circular

Suggested palette:

- `#0A0D13`
- `#111621`
- `#38B8FA`
- `#6F82DD`
- `#FFFFFF`
- success accent: `#45D184`
- warning accent: `#F2B744`

## UI palette

- background: near-black graphite
- primary accent: icy electric blue
- secondary accent: cool steel violet
- success: restrained green
- warning: sparing amber

## Typography

Use system typography with:

- bold rounded display weights for hero moments
- standard body weights for dense gym logging tasks
- strong numeric emphasis for sets, reps, and loads

## Interaction tone

- quiet confidence
- large tap targets
- minimal visual noise
- motion only where it clarifies state
